LNS Library

C++ library implementing 8-bit and 16-bit Logarithmic Number System arithmetic, with exact multiply/divide via exponent addition, spline-based LUT approximation for add/sub and both a hardware compiler header (custom RISC-V instructions) and a software simulation API for benchmarking against float32 and bfloat16.
